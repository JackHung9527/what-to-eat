# 吃什麼？🍜

一個幫你決定「今天要吃哪一家」的個人口袋名單 PWA。不用裝 app、不用註冊、資料完全存在自己手機裡。

👉 **線上使用**：https://jackhung9527.github.io/what-to-eat/

-----

## 為什麼做這個

每次吃飯前都在想「今天吃什麼？」，要嘛選擇障礙、要嘛一直吃同幾家。
這個小工具讓你把喜歡的店家建成口袋名單，按下按鈕隨機抽一家，直接開導航就走。

-----

## 功能特色

- 🍳 **分餐別管理**：早餐 / 午餐 / 晚餐可複選，一家店可以同時是早餐也是午餐
- 🏷️ **分類標籤**：中式、日式、韓式、火鍋、燒烤⋯⋯等 13 種預設分類
- ⭐ **五星評分**：幫每家店打分數，方便之後挑選
- 📍 **點擊導航**：地址直接連到 Apple Maps，一鍵開導航
- 🕐 **營業時間備註**：自由輸入格式，例如「11:00-14:00 / 週一公休」
- 📝 **個人備註**：記錄招牌菜、價位、是否要訂位等資訊
- 🎲 **隨機抽選**：依餐別 + 種類篩選後隨機抽一家，有抽獎動畫
- 📤 **匯出備份**：資料可存成 JSON 檔，防止換手機 / 清資料後遺失
- 📥 **匯入還原**：支援「合併」或「覆蓋」模式
- 🌙 **深色模式**：自動跟隨系統設定
- 📱 **PWA 支援**：加到 iPhone 主畫面後體驗接近原生 app

## 怎麼使用

### 第一次打開

1. Safari 打開 https://jackhung9527.github.io/what-to-eat/
1. 點底部分享按鈕 → **加入主畫面**
1. 回到主畫面，點「吃什麼」icon 開始使用

### 建立口袋名單

1. 底部切換到 **📋 我的名單**
1. 點右上 **＋ 新增店家**
1. 填寫資訊（只有店名是必填，其他都可留空）：
- **店名**：例如「鼎泰豐」
- **餐別**：可複選（早、午、晚）
- **種類**：單選（中式、日式、速食⋯⋯）
- **評分**：1-5 星，再點一次同星數可取消
- **地址**：文字地址，自動轉成 Apple Maps 連結
- **營業時間**：自由文字
- **備註**：自由文字
1. 點 **儲存**

### 隨機抽店家

1. 底部切換到 **🎲 抽店家**
1. 上方篩選條件：
- **餐別**：全部 / 早餐 / 午餐 / 晚餐
- **種類**：全部 / 特定種類
1. 點 **🎲 隨機抽一家**
1. 結果卡會顯示完整資訊，點地址即可開導航

### 編輯或刪除店家

在「我的名單」頁點任一家店，會開啟編輯視窗，可修改或刪除。

## 資料備份（匯出匯入）

### 為什麼要備份

這個 app 的資料存在 iPhone Safari 的 localStorage，以下情況會**消失**：

- 清除 Safari 瀏覽資料
- iOS 長期不使用時自動清理 PWA 資料
- 換手機、重置 iPhone

**建議每幾週或大改清單後匯出備份一次。**

### 匯出步驟

1. **📋 我的名單** 頁 → 點 **📤 匯出**
1. 會產生檔名像 `what-to-eat_20260421.json` 的備份檔
1. iOS 跳出分享選單，可以選：
- **儲存到檔案** → iCloud Drive（推薦，跨裝置可用）
- **郵件** → 寄給自己當備份
- **AirDrop** → 傳到其他 Apple 裝置
- **訊息 / LINE** → 傳給自己或其他人

### 匯入步驟

1. **📋 我的名單** 頁 → 點 **📥 匯入**
1. 選擇之前匯出的 JSON 檔
1. 如果當前已有資料，會問你：
- **輸入 1**：合併（保留現有 + 加入新的，店名重複會略過）
- **輸入 2**：覆蓋（清空現有，全部改用匯入的）
- 其他：取消

### 多裝置「偽同步」

雖然不是自動同步，但配合 iCloud Drive 可以做到類似效果：

1. iPhone 匯出 → 儲存到 iCloud Drive
1. iPad 或新手機打開 app → 匯入 → 從 iCloud Drive 選檔案
1. 每次清單有大變動，重複匯出覆蓋備份檔

## 技術細節

### 使用的技術

- 純 HTML + CSS + JavaScript（單檔）
- 無框架、無外部依賴
- 使用瀏覽器內建 `localStorage` 儲存資料
- 透過 PWA meta 標籤支援加入主畫面
- 匯出優先使用傳統下載，PWA 模式下自動改用 Web Share API

### 檔案結構

```
what-to-eat/
├── index.html   # 主程式（HTML + CSS + JS 全部在一個檔案）
└── README.md    # 這份說明
```

### 資料格式

每家店在 localStorage 中以以下 JSON 結構儲存：

```json
{
  "id": "lx8k2n3mk9p",
  "name": "鼎泰豐",
  "meals": ["lunch", "dinner"],
  "category": "中式",
  "stars": 5,
  "address": "台北市信義區松高路 11 號",
  "hours": "11:00-21:00",
  "note": "招牌小籠包、需排隊"
}
```
